💡 Where do I find a public annual-report PDF?
  • BSE → search the company → "Annual Reports" tab → right-click the latest PDF → "Copy link address".
  • Company IR page → "Investors" → "Financial Reports" / "Annual Reports".
  • SEBI's filings are public but harder to navigate — BSE is usually fastest.
  • Make sure the link ends in .pdf — many IR pages link to an HTML viewer instead of the file itself.
